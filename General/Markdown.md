# Markdown

## Create headings

```markdown
# Some label

## Some label

### Some label

#### Some label

##### Some label

###### Some label
```

## Create a paragraph

```markdown
Some text
```

## Create bold text

```markdown
**Some text**
```

## Create italic text

```markdown
_Some text_
```

## Create a list of items

```markdown
- Some label
- Some label
```

## Create a list of items that are in order

```markdown
1. Some label
2. Some label
```

## Create a link to a URL

```markdown
[Some label](some-url)
```

## Create an embedded image

```markdown
![Some description](some-path)
```

## Create a quote

```markdown
> Some quote
```

## Create a code block

````markdown
```some-language
some-code
```
````
